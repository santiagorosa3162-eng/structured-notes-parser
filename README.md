# Structured Notes Parser

A lightweight frontend utility that transforms unstructured meeting notes into categorized summaries such as action items, decisions, deadlines and participants.

## Overview

Structured Notes Parser analyzes raw meeting text and organizes it into clear, actionable sections. The goal is to demonstrate structured text parsing, classification logic and clean UI rendering using vanilla JavaScript.

## Features

- Sentence parsing and normalization  
- Action item detection  
- Decision and discussion classification  
- Deadline identification  
- Participant extraction  
- Clean structured output  
- Dark professional interface  
- Zero external dependencies  

## Technical Focus

- Pattern-based parsing logic  
- Category mapping system  
- Controlled rendering without duplication  
- Defensive handling of edge cases  
- Lightweight architecture (single HTML file)

## Tech Stack

- HTML  
- CSS  
- Vanilla JavaScript
